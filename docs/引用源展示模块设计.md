# 引用源展示模块设计

> 实现：`CitationList` / `CitationRefRow` / `utils/citation.ts`  
> 协议：SSE `references` 事件 + `CitationRef` 与正文 `[n]` 对齐

## 交互（已实现）

1. **正文上标**：回答中的 `[1][2]` 渲染为灰色圆角数字，可点击  
2. **参考资料区**：「为你找到 N 篇参考资料」列表，默认展开  
3. **双向联动**：点击正文上标 → 滚动并高亮对应行；悬停列表行 → 正文上标高亮  
4. **跳转**：点击资料行 → 知识条目详情（带 `chunk` / `page` query）

## CitationRef 字段

见 `backend/app/schemas/prompt.py` 与 `frontend/src/types/chat.ts`。

新增展示字段：`source_type`、`file_name`、`category`、`updated_at`。

## 正文 linkify

`linkifyCitationMarkers` 将 `[n]` 转为 Markdown 链接 `[n](#cite-n)`，由 CSS 样式化为上标徽章。
